8. Thanks

Thanks

Last but not least we want to say thank you to a bunch of people, communities and projects without which, Fela would have never been possible.

Projects

  • React: For the innovation to handle modern UI.
  • Redux: For the great API design and amazing documentation which was adopted in many ways.
  • Aphrodite: For the simple declarative inline style API.
  • css-in-js: For the great comparison of CSS in JavaScript solutions.

People

  • Christopher "vjeux" Chedeau: For your inspirational talk playing with CSS in JavaScript.
  • Cheng Lou: For your "powerful" talk about abstraction.
  • Lee Byron: For your informative talk about Immutable UI

Community

We also want to thank the whole React and JavaScript community for your open minds to explore new stuff such as CSS in JavaScript solutions.
Also special thanks to every contributor and to everyone who helped discuss the featuers and future of Fela on Gitter.
