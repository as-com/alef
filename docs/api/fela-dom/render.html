6.2.1. render

render(renderer, mountNode)

Renders all cached styles into a DOM node. It also adds a change listener to automatically add newly rendered styles.

Arguments

  1. renderer (Renderer): The renderer providing the styles which are rendered into the mountNode.
  2. mountNode (HTMLElement): DOM node to render your CSS into. In production it must be a <style> element. Though in development, all other HTMLElements will work too, e.g. if you want to render the CSS into a <div> for debugging.

Example

import { createRenderer } from 'fela'
import { render } from 'fela-dom'

// You would most likely add an existing <style>-element
// to your index.html and reference it with a special id
const mountNode = document.getElementById('stylesheet')

const rule = props => ({
  backgroundColor: 'red',
  fontSize: props.size,
  color: 'blue'
})

const renderer = createRenderer()

renderer.render(rule, { size: '12px' }) // => a b c

render(renderer, mountNode)

// automatically adds the rule to the stylesheet
renderer.renderRule(rule, { size: '15px '}) // => a d c
